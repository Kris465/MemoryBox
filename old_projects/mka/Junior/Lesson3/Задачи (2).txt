Задача 1: Угадай число
Описание: Программа загадывает число от 1 до 100, а игрок должен его угадать.

Задача 2: Сумма чисел в списке
Описание: Пользователь вводит числа, пока не введет 'stop'. Программа выводит сумму введенных чисел.

Задача 3: Калькулятор
Описание: Простой калькулятор, который выполняет операции сложения, вычитания, умножения и деления.

Задача 4: Флеш-карты
Описание: Игра с флеш-картами для запоминания слов. Программа показывает слово на английском, пользователь вводит перевод.

Задача 5: Игра в камень-ножницы-бумага
Описание: Игрок играет против компьютера в камень-ножницы-бумага.

Задача 6: Генератор паролей
Описание: Программа генерирует случайный пароль заданной длины.

Задача 7: Список дел
Описание: Программа для создания списка дел с возможностью добавления и удаления задач.

Задача 8: Игра в угадай слово
Описание: Компьютер загадывает слово, а игрок должен угадать его по буквам.

Задача 9: Таблица умножения
Описание: Программа выводит таблицу умножения для заданного числа.

Задача 10: Случайный выбор из списка
Описание: Пользователь вводит список элементов, а программа случайным образом выбирает один из них.
