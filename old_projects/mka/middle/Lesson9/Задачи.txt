▎Задача 1: Супергерои и их силы

Напишите программу, которая запрашивает у пользователя имя супергероя и его суперсилы. Программа должна позволять вводить данные для нескольких супергероев и выводить информацию о всех супергероях в формате: "Супергерой [имя] имеет следующие суперсилы: [список суперсил]". Также добавьте возможность сортировки супергероев по количеству суперсил.

▎Задача 2: Друзья из мультфильма

Напишите программу, которая запрашивает у пользователя информацию о друзьях из мультфильма (имя, любимая еда и возраст). Программа должна позволять вводить данные для нескольких друзей и выводить их информацию в формате: "[имя] (возраст) любит [любимую еду]". Также добавьте возможность фильтрации друзей по возрасту (например, показать только тех, кто старше 10 лет).

▎Задача 3: Приключения Гарри Поттера

Напишите программу, которая запрашивает у пользователя имя волшебника и список заклинаний. Затем программа должна позволять пользователю вводить заклинания для проверки. Если волшебник знает заклинание, программа должна сообщить об этом и добавить заклинание в список "известных" заклинаний. В конце программа должна выводить все известные заклинания волшебника.

▎Задача 4: Дино-коллекция

Напишите программу, которая запрашивает у пользователя информацию о динозаврах (имя, вид и возраст). Программа должна позволять вводить данные для нескольких динозавров и выводить информацию о том, вымер ли динозавр (если возраст больше 65 миллионов лет). Также добавьте возможность подсчета количества вымерших и не вымерших динозавров.

▎Задача 5: Книга о животных

Напишите программу, которая запрашивает у пользователя информацию о домашних животных (имя, вид и возраст). Программа должна позволять вводить данные для нескольких животных и выводить их информацию в формате: "[имя] - это [вид], ему [возраст] лет". Также добавьте возможность фильтрации животных по виду.

▎Задача 6: Фильмография актеров

Создайте программу, которая запрашивает у пользователя имя актера и список фильмов, в которых он снимался. Программа должна позволять вводить данные для нескольких актеров и выводить информацию о всех актерах в формате: "[имя актера] снялся в следующих фильмах: [список фильмов]". Также добавьте возможность поиска актеров по названию фильма.

▎Задача 7: Приключения в мире фэнтези

Напишите программу, которая запрашивает у пользователя имя персонажа и его класс (например, маг, воин, вор) и список умений. Программа должна позволять вводить данные для нескольких персонажей и выводить информацию о всех персонажах в формате: "[имя] - [класс], умеет: [список умений]". Также добавьте возможность сортировки персонажей по классу.

▎Задача 8: Музыкальная коллекция

Создайте программу, которая запрашивает у пользователя информацию о любимых песнях (название, исполнитель и год выпуска). Программа должна позволять вводить данные для нескольких песен и выводить их в формате: "[название] - [исполнитель] ([год])". Также добавьте возможность поиска песен по исполнителю.

▎Задача 9: Кулинарная книга

Напишите программу, которая запрашивает у пользователя название рецепта и список ингредиентов. Программа должна позволять вводить данные для нескольких рецептов и выводить их в формате: "Рецепт [название]: [список ингредиентов]". Также добавьте возможность фильтрации рецептов по наличию определенного ингредиента.

▎Задача 10: Спортсмены и их достижения

Создайте программу, которая запрашивает у пользователя имя спортсмена и его достижения (например, медали, рекорды). Программа должна позволять вводить данные для нескольких спортсменов и выводить их в формате: "[имя спортсмена] достиг следующих успехов: [список достижений]". Также добавьте возможность сортировки спортсменов по количеству достижений.
