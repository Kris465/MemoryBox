## Записки.

Это приложение работает с заметками.

# Задание:

Дописать проект, "Записки", содержащий работу с записками из консоли (можно прочитать все записки, создать одну записку, отредактировать записку, удалить, и прочитать одну записку по ID), записка содержит в себе как минимум 3 поля Id, заголовок и текст, можно добавить дату. Хранение по вашему выбору, можно в текстовом файле (или каждая записка в одном файле, как вам удобнее).
