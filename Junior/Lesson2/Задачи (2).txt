Задача 1: Угадай персонажа
Напишите программу, которая предлагает пользователю угадать персонажа мультфильма. Программа должна использовать словарь с персонажами и подсказками.


Задача 2: Список любимых мультфильмов
Создайте программу, которая позволяет пользователю ввести список своих любимых мультфильмов и затем вывести их.


Задача 3: Персонажи по мультфильму
Напишите программу, которая запрашивает у пользователя название мультфильма и выводит список его персонажей.


Задача 4: Счетчик персонажей
Напишите программу, которая подсчитывает количество персонажей в заданном мультфильме.


Задача 5: Оценка персонажа
Создайте программу, которая позволяет пользователю оценить персонажа от 1 до 5.


Задача 6: Найди любимого персонажа
Напишите программу, которая предлагает пользователю выбрать своего любимого персонажа из списка.


Задача 7: Персонажи с определенной буквы
Создайте программу, которая выводит всех персонажей, начинающихся на заданную букву.


Задача 8: Узнай, кто это!
Напишите программу, которая задает вопросы о персонаже и позволяет пользователю угадывать его.


Задача 9: Персонажи по цвету
Создайте программу, которая позволяет пользователю ввести цвет и выводит всех персонажей этого цвета.


Задача 10: Мультфильм по персонажу
Напишите программу, которая принимает имя персонажа и выводит название мультфильма, в котором он появляется.
