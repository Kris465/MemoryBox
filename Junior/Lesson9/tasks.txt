Задача 1: Приключение Кота

Кот по имени Мурзик собирает мышей. Пользователь вводит количество мышей, которых поймал Мурзик, а затем вводит имена каждой мыши. Напишите программу, которая выводит всех пойманных мышей.


Задача 2: Сбор Яблок

Яблоня-1 дает разные яблоки. Пользователь вводит количество упавших яблок, а затем их цвета. Напишите программу, которая выводит все яблоки и их цвета.


Задача 3: Уроки Совы

Сова-Учительница учит детей. Пользователь вводит количество учеников, а затем их имена. Напишите программу, которая поздравляет каждого ученика с успешным обучением.


Задача 4: Поиски Собак

Собака Шарик ищет своих друзей. Пользователь вводит список друзей-собак. Напишите программу, которая проверяет, есть ли у Шарика друг по имени "Рекс".


Задача 5: Готовим Пирожки

Повар Паша готовит пирожки. Пользователь вводит список ингредиентов. Напишите программу, которая выводит все необходимые ингредиенты и спрашивает, есть ли у повара каждый из них.


Задача 6: Охота за Сокровищами

Пират Капитан Джек ищет сокровища. Пользователь вводит список мест, где он искал. Напишите программу, которая выводит все места и спрашивает, нашел ли он сокровища в каждом из них.


Задача 7: Приветствие Друзей

Ежик Ежик хочет поздороваться со своими друзьями. Пользователь вводит список друзей. Напишите программу, которая выводит приветствие для каждого друга и спрашивает, как дела у каждого из них.


Задача 8: Сбор Цветов
Цветочек Роза собирает цветы. Пользователь вводит количество собранных цветов и их названия. Напишите программу, которая выводит количество собранных цветов и их названия.

Задача 9: Поездка на Поезде

Поезд Экспресс хочет узнать, сколько пассажиров у него на борту. Пользователь вводит количество пассажиров и их имена. Напишите программу, которая выводит количество пассажиров и их имена.

Задача 10: Уборка в Доме

Слоненок убирает в своем доме. Пользователь вводит список вещей, которые нужно убрать. Напишите программу, которая выводит все вещи и спрашивает пользователя, убрал ли он каждую из них.
